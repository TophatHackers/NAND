# NAND
Groundbreaking NAND language based on the universal NAND gate

### Instructions

| **2 bit** | **OPcode** |
|:----------------|:----------------|
| `00`           |`NAND`|
| `01`           |  `STACK` |
| `10`           |`START`|
| `11`           | `END` |

2 bit - OPcode:
00 - NAND
01 - STACK
10 - START
11 - END
